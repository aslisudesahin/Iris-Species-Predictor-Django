# Iris-Species-Predictor-Django
A Django-based web application that predicts Iris flower species using Machine Learning. Features a full CRUD system for dataset management, user authentication, and an interactive prediction interface.
🌸 Iris Species Predictor (Django & ML)
This project is a web-based integration of Machine Learning and Web Development, allowing users to manage Iris flower data and perform real-time species predictions.

🧠 AI & Machine Learning Features
Species Prediction: Uses a trained model to classify Iris flowers based on sepal and petal measurements.

Dataset Management: Interactive dashboard to add, update, delete, and search flower data.

Data Visualization: Clean and responsive UI for presenting classification results.

🛠️ Technical Stack
Backend: Django (Python Framework)

Frontend: HTML5, CSS3 (Custom styling)

Database: SQLite/PostgreSQL integration via Django ORM.

API: Serializers included for data processing.

 How to Run
Clone the repository.

Install dependencies: pip install -r requirements.txt

Run migrations: python manage.py migrate

Start server: python manage.py runserver
